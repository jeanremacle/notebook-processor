# Week 9 — Email Prioritization

## Objective

Build an email prioritization system using LLM prompting techniques.

## Tasks

1. **System Prompt Design** — Write a system prompt that instructs the model
   to classify emails by priority (high/medium/low).

2. **User Prompt Template** — Create a user prompt template that formats
   the email subject and body for classification.

3. **Evaluation** — Run the model on the provided sample emails and
   analyze the results.

## Deliverables

- Completed notebook with all TODO cells filled in
- Brief written analysis in the markdown cells
